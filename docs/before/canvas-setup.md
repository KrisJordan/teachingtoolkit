# Setting Up Canvas

A step-by-step guide to configuring your Canvas course for the semester.

!!! info "Coming Soon"
    This page is under development. Check back soon for detailed Canvas setup instructions.

## Quick Start

1. Access your course from [canvas.unc.edu](https://canvas.unc.edu)
2. Customize your course home page
3. Set up your syllabus and modules
4. Configure your gradebook
5. Publish when ready for students
