# Canvas Tips & Tricks

Make the most of UNC's learning management system with these tips curated from experienced instructors.

## Quick Wins

### Navigation

- **Customize your course menu** — Remove unused items to reduce student confusion
- **Use modules** — Structure content week-by-week or unit-by-unit
- **Set a meaningful home page** — Don't leave the default; create a welcoming landing page

### Communication

- **Announcement settings** — Require students to acknowledge important announcements
- **Inbox filters** — Use filters to manage student messages efficiently
- **Calendar integration** — Help students by keeping all due dates in Canvas calendar

### Grading

- **SpeedGrader shortcuts** — Learn keyboard shortcuts for faster grading
- **Rubrics** — Create reusable rubrics for consistent feedback
- **Comment library** — Save frequently used comments
