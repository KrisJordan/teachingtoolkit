# AI Tool Finder for Instructors

_Last Updated: April 2026_

This page is designed to help UNC instructors choose the right AI tool for the job they are actually trying to do. Start with the teaching task, then check the data constraints.

For a policy-first view of the same landscape, see the [AI Tool Directory](ai-tool-directory.md), which groups tools by UNC support status, activity type, privacy tier, and training-protection language.

!!! warning "Data comes first"

    If you are working with identifiable student grades, feedback, or other FERPA-protected records, start with [Copilot 365 Teaching Tools](copilot-365.md). It is the only AI platform in this toolkit currently described as approved for Tier 2 / FERPA data.

## Start Here by Teaching Task

<div class="tool-chooser-list" markdown>

- **Draft a lesson, activity, or rubric:** Start with [Copilot 365](copilot-365.md) for UNC-supported planning and protected workflows. Try [MagicSchool.ai](magic-school.md) if you want faster form-based generators for rubrics, worksheets, and group work.
- **Grade faster at scale:** Start with [Gradescope](gradescope.md) when many students are producing similar answers and you want answer grouping plus more consistent rubric use.
- **Build a student-facing helper or compare models:** Start with [PromptLab](promptlab.md) when you want a campus-supported AI chat space, side-by-side model comparison, or a simple no-code course agent.
- **Ask questions about readings or course documents:** Start with [NotebookLM](notebooklm.md) when you want source-grounded answers, study guides, and citations back to uploaded materials.
- **Capture meeting notes and action items:** Start with [Zoom AI Companion](https://www.zoom.com/en/products/ai-companion/) when the main need is a meeting recap, action items, or in-meeting assistance.
- **Create visuals for slides, handouts, or course media:** Start with [Adobe Firefly](https://www.adobe.com/products/firefly.html) when you need images, short video assets, or visual concepts for teaching materials.

</div>

## Good Defaults for Common Instructor Needs

- **"I need a rubric draft fast."** Start with [Copilot 365](copilot-365.md) or [MagicSchool.ai](magic-school.md).
- **"I need to grade hundreds of similar responses."** Start with [Gradescope](gradescope.md).
- **"I want a course agent or a safe campus chat workspace for brainstorming."** Start with [PromptLab](promptlab.md).
- **"I want the AI to stay grounded in my uploaded readings and show receipts."** Start with [NotebookLM](notebooklm.md).
- **"I need a meeting recap from office hours or a TA meeting."** Start with Zoom AI Companion.
- **"I need better slide graphics, worksheet visuals, or short media assets."** Start with Adobe Firefly.
