??? note "Video Demo Script Outline (under 10 minutes)"

    **1. Setup the assignment (1 min):** Open a fixed-template PDF assignment in Gradescope. Show that the uploaded template is completely blank and includes a clear final-answer box for each question.

    **2. Choose the grading method (1 min):** In **Grade Submissions**, open a question and select **Form Answer Groups First** instead of grading individually.

    **3. Set the question type (1 min):** Choose **Math Fill-in-the-blank** or **Text Fill-in-the-blank** and briefly explain that these are the question types where Gradescope can suggest groups automatically.

    **4. Review AI-suggested groups (2 min):** Show the proposed groups, rename one, merge two similar groups, and remove one answer that does not belong.

    **5. Grade by group (2 min):** Apply rubric items to one group and show how the score and comment apply to every student in that group.

    **6. Handle reasoning and edge cases (2 min):** Open the submissions within one group to show how you can still review each student's work or reasoning one-by-one when needed.

    **7. Wrap-up (1 min):** Recap: blank template, clear answer regions, AI suggests groups, instructor confirms groups, one rubric decision can grade many students at once.
