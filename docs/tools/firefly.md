Adobe Firefly is Adobe's generative media tool for creating images, editing images with generative fill, and experimenting with visual concepts for teaching materials. Its current UNC-accessible image models are better suited to rough illustrative exploration than to accurate instructional graphics.

!!! tip "Recommended alternative for most instructors"

    For image generation in everyday teaching workflows, start with [Copilot 365](copilot-365.md) in UNC's Microsoft 365 environment. Copilot has strong image generation tools, is UNC supported, and is available with an active Onyen without a separate access request.

Use Firefly only for public or low-sensitivity teaching materials, such as decorative images, mood boards, simple image edits, or early visual concepts. Adobe trains Firefly on licensed and public-domain content, which addresses copyright questions. Even so, Firefly should not be trusted to create accurate diagrams, labels, or slide content for teaching.

Eligible instructional faculty and instructional staff can request Creative Cloud at no cost if access is not already active.

## Access for Instructors

### Request Creative Cloud

UNC instructors who do not already have Creative Cloud access can request it through the UNC service request:

1. Open the [Adobe Creative Cloud Service Request](https://tdx.unc.edu/TDClient/33/Portal/Requests/Service/61/Adobe-Creative-Cloud)
2. Sign in with your Onyen if prompted
3. Click **Request Creative Cloud**
4. Fill out the requested customer information, including who the request is for, account/department, and preferred contact method
5. Select the appropriate user type
6. Describe your instructional use if asked
7. Submit the request

Requests can take up to **5 business days** to complete. If you already have Creative Cloud access and only need to use Firefly, you can go directly to Firefly.

### Use Firefly After Access Is Active

Once your Creative Cloud access is active:

1. Go to [firefly.adobe.com](https://firefly.adobe.com/)
2. Sign in with your primary UNC-Chapel Hill email address
3. If prompted, choose **Company or School Account**
4. Complete UNC single sign-on and Duo verification
5. Choose the Firefly feature you want to use
6. Enter prompts that use general course concepts or other public or low-sensitivity teaching content
7. Download the result for review before using it in course materials

Firefly is also integrated into some Adobe desktop and web apps. If you want those integrations, install Creative Cloud following UNC's Adobe Creative Cloud install guide. Firefly's browser version does not require installing the full Creative Cloud desktop app.

### Use Image 4 or Image 3

With standard UNC instructor access, the available non-premium Firefly image models are **Firefly Image 4** and **Firefly Image 3**. Start with Image 4. Use Image 3 only as a quick comparison if Image 4 gives you an odd result.

| Model | Practical guidance |
|---|---|
| Firefly Image 4 | Best of the available non-premium Firefly options. Use it only for low-stakes illustrative images, simple visual concepts, and basic photo-style objects that do not need accurate labels, text, diagrams, or disciplinary detail. |
| Firefly Image 3 | Older than Image 4. Use only as a fallback or comparison model for quick ideation or style exploration. |

In Firefly's model menu, a purple crown marks a premium model or feature. Many Text to Image model choices, such as Firefly Image 4 Ultra, and partner models are premium generative features.

If you select a premium model, Firefly may show a red message asking you to contact your admin to buy a plan with premium feature access. That message means the feature is outside standard UNC instructor access. Premium Firefly features are not supported through the standard UNC Creative Cloud license and require departmental licensing before use.

For state of the art image generation without a Creative Cloud request or departmental Firefly licensing, use [Copilot 365](copilot-365.md). [Copilot Chat](https://its.unc.edu/ai/copilot/) is a centrally supported generative AI tool available to users with an active Onyen.

## Teaching Use Cases

### Explore Low-Stakes Visual Concepts

Use Firefly to explore decorative or metaphorical images that do not need to teach factual relationships. It can be helpful when you want a rough visual direction and the image will be reviewed, edited, or recreated before it becomes part of a course artifact.

### Edit Existing Teaching Images

Generative Fill can help remove distracting background elements, extend a non-sensitive image, or create a simple variation of an image you have rights to use.

### Prototype Visual Examples

Firefly can help brainstorm visual metaphors, scene compositions, or alternative versions of an example before you decide whether to create a more polished asset elsewhere.

!!! warning "Not Recommended for Slides or Diagrams"

    Do not use Firefly Image 4 or Image 3 at this time to generate slides, flowcharts, diagrams, labeled scientific cycles, process graphics, or other text-heavy instructional visuals. These models can produce polished-looking images with garbled text, invented labels, incorrect relationships, and misleading visual structure. For teaching visuals that need to explain a concept more accurately, use [Copilot 365](copilot-365.md), PowerPoint, or another workflow where you can build and verify the structure directly.

## Data Privacy

Firefly image-generation prompts for teaching should not need FERPA-protected student records. It is not capable of producing data visualizations from raw input data. You must not use sensitive information in Adobe cloud-based storage. For teaching work, keep student education records out of Firefly prompts, uploads, generated assets, and Adobe cloud storage.

## Notes About Firefly Outputs

Current Firefly generative AI models are trained on licensed content, such as Adobe Stock, and public domain content where copyright has expired. The first commercially released Firefly model was trained on Adobe Stock images, openly licensed content, and public domain content where copyright has expired. Firefly can apply content credentials to some AI-generated or AI-edited content, making it easier to identify that AI was used.

As with any image generator, review outputs carefully before using them in class. Check for visual inaccuracies, cultural stereotypes, incorrect text, accessibility problems, and misleading representations of people, places, or course concepts. Do not treat Firefly-generated diagrams or slide-like images as accurate teaching materials without rebuilding and verifying them.
