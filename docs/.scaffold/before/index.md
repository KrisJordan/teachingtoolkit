## Timeline Overview

<div class="timeline-item" markdown>
### 4-6 Weeks Before

- Review and update syllabus
- Order textbooks and materials
- Request any needed classroom technology
- Plan major assignments and due dates
</div>

<div class="timeline-item" markdown>
### 2-3 Weeks Before

- Set up Canvas course site
- Create first week's content and activities
- Prepare welcome announcement
- Test any technology you'll be using
</div>

<div class="timeline-item" markdown>
### Week Before

- Publish Canvas course to students
- Send welcome email/announcement
- Verify classroom access and setup
- Review roster and note any accommodations
</div>
