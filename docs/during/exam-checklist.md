# Exam Preparation Checklist

A comprehensive checklist for creating, administering, and returning exams. Whether you're giving your first exam or your hundredth, this checklist helps ensure nothing gets overlooked.

---

## :material-pencil: Creating the Exam

### Content Planning

- [ ] Review learning objectives covered since last assessment
- [ ] Ensure questions align with what was taught and practiced
- [ ] Include a mix of question types (if appropriate for your discipline)
- [ ] Balance difficulty levels (some confidence-builders, some challenges)
- [ ] Check that point values add up correctly
- [ ] Estimate time needed; allow students adequate time

### Question Quality

- [ ] Write clear, unambiguous questions
- [ ] Avoid "trick" questions or unnecessarily complex wording
- [ ] For multiple choice: ensure all distractors are plausible
- [ ] For short answer/essay: clarify expected length and format
- [ ] Check for unintended clues (e.g., grammatical hints in MC questions)
- [ ] Have a colleague or TA review if possible

### Logistics

- [ ] Create clear instructions at the top of the exam
- [ ] Include point values for each question or section
- [ ] Leave adequate space for answers (or specify separate answer sheet)
- [ ] Number pages and include "continued on next page" prompts
- [ ] Create a cover page with name field and honor pledge if required

---

## :material-account-multiple: Accommodations

!!! warning "Don't forget!"
    Students with ARS accommodations need advance notice. Don't wait until the last minute.

- [ ] Identify students with ARS accommodations early in the semester
- [ ] Schedule accommodated testing with ARS (typically 5-7 days ahead)
- [ ] Prepare alternate format exams if needed (large print, digital, etc.)
- [ ] Communicate with students about their testing arrangements
- [ ] Plan for how accommodated and regular exams will be administered simultaneously

---

## :material-clock: Before Exam Day

### One Week Before

- [ ] Send announcement with exam details (date, time, location, format)
- [ ] Clarify what materials are permitted (notes, calculator, etc.)
- [ ] Share any review materials or practice problems
- [ ] Hold review session or extended office hours
- [ ] Finalize and print exam (or set up in Canvas for digital)

### Day Before

- [ ] Confirm room setup and any needed technology
- [ ] Print extra copies in case of issues
- [ ] Prepare seating chart if using assigned seating
- [ ] If using TAs: brief them on proctoring expectations
- [ ] Get a good night's sleep yourself!

---

## :material-school: Exam Day

### Before Students Arrive

- [ ] Arrive early to set up the room
- [ ] Test projector/screen if displaying instructions
- [ ] Arrange seating to minimize cheating opportunities
- [ ] Write start time, end time, and any key info on board
- [ ] Have blank scratch paper available if needed

### During the Exam

- [ ] Announce start time and how much time remains periodically
- [ ] Walk around the room periodically (but don't hover)
- [ ] Be available for questions but minimize disruption
- [ ] Note any issues or questions that arise (for future reference)
- [ ] Handle suspected academic integrity issues calmly and discreetly
- [ ] Give 15-minute, 5-minute, and 1-minute warnings

### Collecting Exams

- [ ] Ensure all exams are accounted for
- [ ] Collect any scratch paper if required
- [ ] Keep exams secure (never leave unattended)
- [ ] Thank students for their effort

---

## :material-check-all: After the Exam

### Grading

- [ ] Create or finalize rubric/answer key before grading
- [ ] If using TAs: meet to norm grading expectations
- [ ] Grade all responses to one question before moving to the next
- [ ] Note common errors or misconceptions for class discussion
- [ ] Double-check grade calculations
- [ ] Enter grades in Canvas promptly

### Returning Exams

- [ ] Return exams within a reasonable timeframe (1-2 weeks)
- [ ] Decide how to return: in class, office hours, Canvas (for digital)
- [ ] Plan for how students can review and discuss their performance
- [ ] Post solutions or review key points in class
- [ ] Be prepared for grade disputes; set clear regrade policies

### Reflection

- [ ] Note questions that were too easy or too hard
- [ ] Identify concepts students struggled with (reteach if needed)
- [ ] Save data for course improvement planning
- [ ] File exam securely for your records
